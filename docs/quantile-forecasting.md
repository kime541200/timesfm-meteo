# Quantile Forecasting

利用分位數預測（Quantile Forecasting）來評估氣溫的不確定性，核心在於**從「單一數值預測」轉變為「機率分佈預測」**。與傳統只輸出單一預估值（點預測）的模型不同，分位數預測能直接輸出多個特定分位數（例如 10%、50%、90% 分位數）的預測結果，藉此提供不確定性的估計，且無需依賴繁複的多次抽樣運算。

在氣溫預測的實際應用上，您可以透過以下方式來評估不確定性：

1.  **建立氣溫的預測區間（Prediction Intervals）：** 您可以同時預測同一時間點的低分位數（如 10%）與高分位數（如 90%）。這兩個數值之間的範圍，就構成了一個 80% 的預測區間，用來表示未來實際氣溫可能落入的範圍。
2.  **透過「區間寬度」直觀衡量風險：** 
    *   如果高低分位數之間的**差距（帶寬）很寬**，代表模型研判未來的氣溫變數很大（例如即將有鋒面經過或極端氣候），這時氣溫預測的「不確定性極高」。
    *   反之，若高低分位數的**差距很窄**，則表示模型對該時段的氣溫非常有把握，預測的「不確定性極低」。
3.  **TimesFM 的分位數輸出：** 若所用 TimesFM 版本支援分位數預測頭（Quantile head），可直接輸出多個分位數，用來畫出未來每日氣溫的機率帶，並觀察極端高溫或低溫風險。

總結來說，分位數預測能讓您在觀看氣溫預測時，不僅知道「最可能發生的溫度（通常為 50% 分位數或中位數）」，還能明確掌握「氣溫最多可能偏離多少」，是量化氣候預測風險與不確定性非常有效的方式。
